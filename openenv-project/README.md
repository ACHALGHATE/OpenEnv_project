# AI Work Assistant (Top 1% OpenEnv Submission)

## 🚀 Overview
This project simulates real-world workplace tasks including email triage, customer support, and meeting scheduling. It is designed to train AI agents in practical decision-making scenarios.

## 🧠 Tasks
- Email triage (priority classification)
- Customer support (natural language response)
- Scheduling (conflict resolution)

## 🎯 Key Features
- Multi-task environment
- Continuous reward shaping
- Deterministic grading
- LLM-driven agent

## ⚙️ Run
```bash
docker build -t openenv .
docker run openenv
```
